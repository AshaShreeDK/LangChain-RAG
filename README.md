LangChain-RAG


Production-grade Retrieval-Augmented Generation (RAG) system using:
- LangChain (LCEL)
- FAISS (versioned indexing)
- HuggingFace embeddings
- Groq LLM

Features:
- Explicit ingestion and indexing
- Versioned FAISS indexes with CURRENT pointer
- Chunk inspection and auditing utilities
- Domain-aware retrieval with source attribution
